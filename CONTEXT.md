# Project: GrafColor Premium Landing Page

## Project Summary
Landing Page de alta conversão para a gráfica GrafColor. 
Design: Dark Mode, gradientes magenta/purple, estética high-tech.
Objetivo: Captação de leads via WhatsApp e Formulário.

## Tech Stack
- Framework: Astro 5.x (Static Site Generation)
- Styling: Tailwind CSS
- Animations: Framer Motion (Scroll animations)
- Icons: Lucide React
- Fonts: Inter ou Sora (conforme mockup)

## Functional Scope
- Navigation: Sticky menu com links internos (âncoras).
- Hero: Título impactante + Imagem da impressora com efeito de brilho.
- Services: Grid com 6 cards (Efeito glassmorphism).
- Product List: Lista categorizada em 3 colunas.
- Testimonials: Slider ou Grid de depoimentos com estrelas e avatar.
- About: Seção institucional com CTA flutuante para WhatsApp.
- Contact: Formulário de orçamento funcional + Rodapé completo.

## Design Tokens (Tailwind)
- Background: #0a0a0a (Deep Black)
- Primary: #ff00ff (Magenta Neon)
- Secondary: #7000ff (Electric Purple)
- Surface: rgba(255, 255, 255, 0.05) (Glassmorphism)
